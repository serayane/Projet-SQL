## Présentation
Ce projet consiste à concevoir et exploiter une base de données relationnelle
pour la gestion d’un service de location de véhicules.

Le travail est réalisé en groupe, avec une répartition claire des tâches
afin de permettre un développement indépendant.

## Répartition du travail
- **Kenza** : analyse et modélisation (Merise – MCD, MLD)
- **Sofia** : création de la base PostgreSQL et scripts SQL
- **Rayane** : requêtes SQL avancées, documentation et dépôt Git

## Tables principales
clients, vehicules, locations, paiements, maintenance, stations
